# delovye-linii-mcp

MCP-сервер для API Деловых Линий (Dellin) — расчёт стоимости, справочник городов, отслеживание.

## Возможности (3 инструмента)

| Инструмент | Описание |
|---|---|
| `calculate` | Расчёт стоимости и сроков доставки |
| `get_cities` | Поиск городов в справочнике |
| `track` | Отслеживание заказа по номеру |

## Быстрый старт

```json
{
  "mcpServers": {
    "dellin": {
      "command": "npx",
      "args": ["-y", "@theyahia/delovye-linii-mcp"],
      "env": {
        "DELLIN_API_KEY": "<YOUR_API_KEY>"
      }
    }
  }
}
```

## Переменные окружения

| Переменная | Обязательная | Описание |
|---|---|---|
| `DELLIN_API_KEY` | Да | API-ключ из кабинета разработчика: https://dev.dellin.ru/ |
