# /skill-track — Отследить груз

Отслеживание статуса груза по номеру накладной Деловых Линий.

## Алгоритм

1. Вызови track с номером накладной
2. Покажи текущий статус, маршрут и ключевые даты
3. Если есть документы — покажи список с их статусами

## Формат вывода

- Номер заказа
- Текущий статус
- Маршрут: Город отправления -> Город назначения
- Даты: заказ, отправка, прибытие, доставка

## Примеры

    /skill-track 1234567890
    /skill-track DL-2025-001
