# /skill-calculate — Расчёт стоимости грузоперевозки

Расчёт стоимости и сроков доставки Деловыми Линиями по маршруту.

## Алгоритм

1. Вызови get_cities для поиска ID города отправления
2. Вызови get_cities для поиска ID города назначения
3. Вызови calculate с ID городов, весом и габаритами (если указаны)
4. Покажи все варианты доставки: стоимость, сроки, тип перевозки
5. Если найдены терминалы — вызови get_terminals и укажи адреса

## Формат вывода

Таблица с колонками: Тип | Стоимость (руб) | Срок (дней)

## Примеры

    /skill-calculate Москва Новосибирск 100
    /skill-calculate Петербург Краснодар 50 1.2 0.8 0.6
