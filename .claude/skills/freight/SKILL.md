# /freight — Грузоперевозка Деловыми Линиями

## Алгоритм

1. Вызови get_cities для поиска ID городов отправления и назначения
2. Вызови calculate с ID городов и параметрами груза
3. Покажи варианты доставки со стоимостью и сроками

## Примеры

    /freight Москва Новосибирск 100
    /freight Петербург Краснодар 50
